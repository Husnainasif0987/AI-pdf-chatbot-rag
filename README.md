# AI PDF Chatbot (RAG)

This project is a Retrieval-Augmented Generation (RAG) chatbot that allows users to upload a PDF and ask questions about it.

The system retrieves relevant information from the document and generates answers using a Large Language Model.

## Features

- Upload PDF documents
- Ask questions from documents
- AI generates contextual answers
- Fast vector search

## Technologies

- Python
- Streamlit
- LangChain
- OpenAI API
- FAISS

## Installation

Clone repository

git clone https://github.com/yourusername/ai-pdf-chatbot-rag.git

Install dependencies

pip install -r requirements.txt

Run the project

streamlit run app.py

## Example Use Case

Upload a research paper, book, or company document and ask questions to quickly get insights.
