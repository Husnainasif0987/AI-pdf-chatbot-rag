# AI PDF Chatbot (RAG)

This project is a Retrieval-Augmented Generation (RAG) chatbot that answers questions from PDF documents using LangChain and OpenAI.

## Features
- Upload PDF documents
- Ask questions from documents
- AI generates contextual answers

## Technologies
- Python
- LangChain
- OpenAI API
- FAISS

## How to Run

1. Install requirements
pip install -r requirements.txt

2. Run project
python app.py
